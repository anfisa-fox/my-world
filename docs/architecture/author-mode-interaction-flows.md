# Author Mode Interaction Flows

Version: 0.1 (Working Draft)

---

# Назначение документа

Документ описывает пользовательские сценарии Author Mode.

Он не определяет техническую реализацию.

Его задача — зафиксировать последовательность действий автора и ожидаемое поведение системы.

В рамках MVP описываются только сценарии, необходимые для проверки основной продуктовой гипотезы.

---

# Основная гипотеза

> Автору должно хотеться создавать новый контент прямо внутри своего мира.

---

# Flow 1 — Возвращение домой

```
Главная страница

↓

Hidden Door (✦)

↓

Мир спрашивает:

«Это ты?»

↓

Автор вводит Studio Secret

↓

Мир отвечает:

«Я узнал тебя.»

↓

Author Mode становится активным

↓

Автор остаётся на той же странице
```

### Принципы

* отсутствует переход на отдельную страницу;
* отсутствует Dashboard;
* отсутствует административная панель;
* мир остаётся визуально тем же самым.

---

# Flow 2 — Создание новой истории

```
Author Mode

↓

Переход в Wall

↓

Нажатие ＋

↓

Форма новой истории

↓

Название

↓

Текст

↓

Публикация

↓

Мир сообщает:

«Я уже отправил новую историю на публикацию.»

↓

Запускается автоматический GitHub Commit

↓

Запускается Cloudflare Deploy

↓

Через некоторое время история появляется на опубликованном сайте
```

### Принципы

Автор не выполняет технических действий.

GitHub, Cloudflare и процесс публикации остаются внутренней реализацией системы.

---

# Flow 3 — Завершение работы

```
Автор продолжает исследовать свой мир

или

Закрывает сайт
```

Author Mode не требует отдельного сценария выхода.

При завершении пользовательской сессии режим автора завершается естественным образом.

---

# Не входит в MVP

Следующие сценарии сознательно исключены из текущей проверки гипотезы:

* редактирование публикаций;
* удаление публикаций;
* создание рисунков;
* создание персонажей;
* загрузка изображений;
* настройка главной страницы (World Showcase);
* предпросмотр публикаций;
* черновики.

Все перечисленные сценарии рассматриваются после анализа результатов первой пользовательской демонстрации.

---

# Definition of Success

После первого знакомства с Author Mode автор самостоятельно возвращается в свой мир и публикует новую историю без посторонней помощи.

Именно этот сценарий является основным критерием успешности MVP.
